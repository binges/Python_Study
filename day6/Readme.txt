目录结构：
bing
    start.py    主程序，运行此文件开始游戏
conf
    config.py   配置文件，可以设置游戏难度，守门员人数和回合数
core
    main.py     类文件



代码模拟的是一个点球游戏
使用random函数
一个点球人（玩家），一个或多个守门员。
    玩家输入一个数字，和守门员随机出来的数字做对比，如果玩家输入的数字在守门员随机出来的数组内，代表守门员把球守住了，守门员加1分，否则玩家得1分
    守门员的个数，来确定每次随机出几个数字。代表守门员守的方位
    游戏难度系数，来确定游戏的难度，最小也是最简单为2。代码实现是这个难度等级来确定随机数的最大值，比如难道系数为5就随机生成1-5之间的数字。（难道系数减2不能小于等于守门员的个数）
    回合数，可以定义玩家点球次数
